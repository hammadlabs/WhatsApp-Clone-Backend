# Chat Application

## WhatsApp Chat Application Clone

#### Discription

A Backend App for realtime messaging application inspired by Whatsapp.This project is build to practice backend development concepts Including Authentication, Authorization, Validations, Socket.io, REST API's and database modeling and communication.

## Core Funcanility

- Login
- Signup
- Refresh Token
- User Profile
- User Conversations
- Conversation Messages
- Conversation Participents

### Tech Stack

- **Backend:** Node.js & Express
- **Database:** PostgreSQL
- **ORM:** Prisma
